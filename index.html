INTRODUCCIÓN

Bienvenidos a este OVI enfocado a la introducción al diseño de sitios web.
Una página web, o página electrónica, página digital, o ciberpágina es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes y muchas otras cosas, adaptada para la llamada World Wide Web (WWW) y que puede ser accedida mediante un navegador web. Esta información se encuentra generalmente en formato HTML o XHTML, y puede proporcionar acceso a otras páginas web mediante enlaces de hipertexto. Frecuentemente también incluyen otros recursos como pueden ser hojas de estilo en cascada, guiones (scripts), imágenes digitales, entre otros. Las páginas web pueden estar almacenadas en un equipo local o en un servidor web remoto. El servidor web puede restringir el acceso únicamente a redes privadas, por ejemplo, en una intranet corporativa, o puede publicar las páginas en la World Wide Web. El acceso a las páginas web es realizado mediante una transferencia desde servidores, utilizando el protocolo de transferencia de hipertexto (HTTP).

Los entornos web son cada vez más importantes, debido a que han facilitado significativamente el intercambio y distribución del conocimiento, convirtiéndose en herramientas de apoyo para que los usuarios con pocos conocimientos puedan crear y gestionar contenidos con relativa facilidad. Por consiguiente, los diseñadores de estos entornos web deben tomar en cuenta un mayor número de criterios durante el proceso de diseño, con el objetivo de satisfacer las necesidades del usuario en particular.

Objetivo general: Planificar, diseñar, implementar y mantener un sitio web, teniendo en cuenta diferentes aspectos como diseño gráfico web, diseños de interfaz y experiencia del usuario, interacción de medios, y la optimización de motores de

Objetivo específico 1: Realizar un boceto o prediseño, tener un orden claro sobre el diseño del sitio, como el diseño visual de la información que se desea editar, distribuir textos, gráficos, vínculos a otros documentos y otros objetos multimedia.

Objetivo específico 2: Estructuración y relación jerárquica de las páginas del sitio web, usando el lenguaje de marcación de hipertexto o HTML.

Objetivo específico 3: Optimizar la estructura del contenido para mejorar la posición en que aparece la página en los motores de búsqueda.
